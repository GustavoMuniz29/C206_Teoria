//Anotações:
    -> Encapsulamento de dados:

        - Modificadores de acesso:

            public: acesso liberado aos recursos
            private: acesso interno da própria classe
            interface: public
            implementação: private (atributos e corpo de métodos)

    -> Getters and Setters:

       Get: permite pegar a informação de uma variavel private
       Set: permite alterar a informação de uma variavel private

    -> Construtores:

        - Pseudo Método
            |_ Não retorna valores
            |_ Chamado uma única vez (Na hora do "new")
            |_ Tem o mesmo nome da CLASSE

    -> Atributo de Classe:

        - Infomação/Atributo de uma classe que todos os objetos conseguem acessar
